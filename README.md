# Codex
World of Warcraft (1.12) addon which displays mob spells on mouseover.

Spell information displayed:
- Name (ex: Fireball)
  - Color corresponding to the spell's school (ex: Physical, Holy, Arcane, ..)
- Subname (ex: Rank 1)
- Dispel type (Poison, Curse, Magic, Disease)
